## Getting started

Clone the repo

```bash
$ git clone https://github.com/mohole/react-workshop
```

change to project folder

```bash
$ cd react-workshop
```

install the dependencies

```bash
$ npm ci
```

start the local development server

```bash
$ npm run dev
```

you can now check your application at `http://localhost:3000`.
